# YouTube Audience Engagement Analysis

An NLP-powered analytics system designed to analyze YouTube audience engagement through sentiment analysis, topic modeling, named entity recognition, and comment intelligence workflows.

---

## Overview

This project analyzes YouTube comments to extract audience sentiment, engagement trends, discussion topics, and named entities using Natural Language Processing (NLP) techniques.

The system automates comment extraction, performs text preprocessing, generates audience insights, and visualizes engagement patterns through interactive analytics workflows.

---

## Key Features

- Automated YouTube comment extraction
- Sentiment analysis on audience comments
- Named Entity Recognition (NER)
- TF-IDF keyword extraction
- LDA topic modeling
- Interactive Streamlit dashboard
- Audience engagement analytics
- Data visualization workflows
